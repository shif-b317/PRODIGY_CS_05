# PRODIGY_CS_05 - Network Packet Analyzer

## Description
This project is a basic Network Packet Analyzer developed using Python and Scapy. The program captures network packets and displays important packet information such as protocol type and packet length while masking sensitive IP information for privacy.

## Features
- Captures live network packets
- Detects protocol types:
  - TCP
  - UDP
  - ICMP
- Displays packet length
- Logs packet details to a file
- Timestamp support
- Privacy-friendly masked IP addresses

## Technologies Used
- Python
- Scapy
